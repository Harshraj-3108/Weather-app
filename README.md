# 🌤️ Weather App

A simple and responsive weather application that displays real-time weather details for any city.

## 🚀 Features
- Search weather by city name
- Shows temperature, humidity, wind speed, and "feels like"
- Clean and modern UI
- Enter key support for quick search
- Error handling for invalid input

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🌐 API Used
- wttr.in (Free weather API, no API key required)

## 📸 How it Works
1. User enters a city name
2. App fetches data from the API
3. Weather details are displayed dynamically
